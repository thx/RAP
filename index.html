RAP is a GUI interface management tool

By analyzing structures of interface, RAP can generate mock data, validate real back-end interface. Around with interface definition, RAP supplys lots of automation tools to improve our efficiencies of team work!

Features

Easily edit and share

GUI edit, nice version control, support lots of format with import/export tools. Defining interface can't be easier in RAP!

More

Mock Service

Based on interface doc, RAP generate MOCK service, support complex MOCK logic.

More

Automation toolbox

Except for tools provided by RAP team, we continuously add new tools by supporting Blueprint API standards.
